# Getting Started

## Installation

You can install TheSecretJuice with a single command:

```powershell
iwr https://raw.githubusercontent.com/mini-page/TheSecretJuice/main/install.ps1 | iex
```

## Quick Start

Once installed, you can start using the enhanced tools immediately. For example, to use the enhanced `yt-dlp` wrapper, simply run:

```powershell
yt-dlp
```

This will launch an interactive menu to guide you through the download process.
