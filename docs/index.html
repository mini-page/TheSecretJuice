Option 2: Add to PowerShell Profile

Frequently Asked Questions

Everything you need to know

Does this replace my existing CLI tools?

No! TheSecretJuice wraps your existing tools with enhanced interfaces. The original commands still work exactly as before. It's purely additive.

What if I prefer the original CLI?

That's fine! Just use the original command with flags. TheSecretJuice only activates when you run the command without arguments. You can also unload specific enhancements from your profile.

Is it safe to use?

Yes! It's open source (MIT license), all code is visible on GitHub, and it has zero telemetry. It only wraps your existing tools and doesn't send any data anywhere.

Does it work on Linux/macOS?

Yes! As long as you have PowerShell Core 7+ installed. TheSecretJuice is cross-platform and works on Windows, Linux, and macOS.

Can I request a tool to be enhanced?

Absolutely! Open an issue on GitHub with your tool suggestion. Better yet, contribute the enhancement yourself - we provide templates and guidelines!

Does it slow down my tools?

Negligible impact! TheSecretJuice scripts are lightweight and only add a few milliseconds of overhead. The interactive menus actually save you time overall.
